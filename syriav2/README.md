نظام سوريا V2.0 - نظام إدارة إعادة الإعمار

نظرة عامة:
سوريا V2.0 هو نظام إلكتروني متكامل لإدارة وتنسيق عملية إعادة إعمار سوريا بعد سنوات الحرب. تم تصميم النظام ليكون منصة شاملة تجمع بين المواطنين والمنظمات والمتعهدين والإداريين لتحقيق إدارة فعالة وشفافة لمشاريع إعادة الإعمار.

الأهداف الرئيسية:
توفير نظام مركزي لإدارة مشاريع إعادة الإعمار -

تمكين المواطنين من المشاركة في تحديد أولويات - المشاريع 

تسهيل عملية التنسيق بين الجهات المختلفة -

رفع مستوى الشفافية في إدارة المشاريع -

توثيق مراحل التنفيذ والتقدم -

الميزات الرئيسية:
نظام المستخدمين متعدد الأدوار -
يدعم النظام أربعة أنواع من المستخدمين، لكل دور - صلاحيات ومهام محددة.

إدارة المشاريع المتكاملة:
تصنيف المشاريع إلى ثلاثة أنواع: إعادة إعمار شاملة، دعم مجتمعي، تنمية إنشائية جديدة -

تتبع حالة المشاريع: قيد الانتظار، قيد التنفيذ، مكتملة، ملغاة -

خرائط تفاعلية لتحديد مواقع المشاريع -

معرض صور لكل مشروع (4 صور كحد أقصى) -

لوحات التحكم والإحصائيات:
إحصائيات حية للمشاريع والمستخدمين -

رسومات بيانية للتكاليف والمساحات -

تقارير مفصلة عن سير العمل -

نظام أمني متطور:
مصادقة آمنة للمستخدمين -

صلاحيات وصول محددة لكل دور -

حذف منطقي للمستخدمين والمشاريع -

تسجيل تلقائي للعمليات المهمة -

الدليل التقني:
أنماط التصميم المستخدمة
Factory Pattern
يستخدم في projects/factories.py لإنشاء أنواع مختلفة من المشاريع:

ComprehensiveProjectFactory: للمشاريع الشاملة

CommunityProjectFactory: للمشاريع المجتمعية

DevelopmentProjectFactory: للمشاريع الإنشائية الجديدة

Repository Pattern
يستخدم في accounts/repositories.py لفصل منطق الوصول للبيانات عن الطبقات الأخرى.

قواعد البيانات
نموذج المستخدم (User)
username: اسم المستخدم (فريد)

role: الدور (مواطن، منظمة، متعهد، مدير النظام)

phone_number: رقم الهاتف

document: وثيقة الهوية (للمنظمة والمتعهد)

is_deleted: علامة الحذف المنطقي

نموذج المشروع (Project)
name: اسم المشروع

project_type: نوع المشروع

description: الوصف التفصيلي

damage_type: نوع الضرر

address: العنوان

governorate: المحافظة

latitude, longitude: الإحداثيات الجغرافية

image1, image2, image3, image4: صور المشروع

estimated_cost: التكلفة المقدرة

duration: المدة المقدرة

area: المساحة

status: حالة المشروع

contractor: المتعهد المعين
